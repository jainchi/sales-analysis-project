# Sales Data Analysis Project

## Objective
Analyze sales data to identify top products, customers, and trends.

## Tools Used
- Python
- Pandas

## Key Insights
- Top-selling product: Mutton Biryani
- Highest spending customer: ID 8
- Sales peak on weekends (Friday–Sunday)
- Lowest activity on Tuesday and Wednesday

## Conclusion
Performed data cleaning, aggregation, and time-based analysis using pandas.
